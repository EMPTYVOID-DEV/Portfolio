**EnvTracker: Multi-Tenant SaaS Platform Modernization**

Envtracker is a SAAS solution that streamlines project management, development, bug tracking and accelerates the release and delivery process. It has been designed to improve time to market by providing real-time visibility, faster deployment and improved focus and productivity for teams. With an intuitive and user-friendly interface. Envtracker allows different teams to collaborate seamlessly and effectively. It also integrates popular tools such as Jira, Git, Jenkins, GitLab.

My work involved end-to-end redesign and evolution of the project into a scalable, multi-tenant SaaS system, supporting real-time deployment tracking, analytics, and enterprise-grade access control.

🔑 **Key Highlights:**

1. **SaaS Architecture Transformation:** Led the migration from a monolithic system to a multi-tenant architecture with clear control plane / application plane separation, enabling scalable tenant onboarding and tier-based deployments.
2. **Tenant Isolation Strategy:** Designed and implemented PostgreSQL schema-based isolation for Free/Pro tenants and full-stack isolated environments for Enterprise clients.
3. **Frontend Re-Architecture:** Rebuilt the entire frontend from React 17 to SvelteKit, introducing a modular, maintainable architecture with improved performance and developer experience.
4. **Backend Refactoring:** Decoupled the core Spring application into a dedicated worker-based architecture to handle asynchronous jobs, external integrations, and background processing.
5. **Real-Time Deployment System:** Engineered a live deployment tracking system using RabbitMQ and Server-Sent Events, supporting 1000+ monthly deployments with real-time status updates.
6. **Advanced Access Control:** Implemented a hybrid RBAC/ABAC/ReBAC authorization model with fine-grained permissions across organizations, projects, teams, and environments.
7. **Enterprise SSO Integration:** Integrated OAuth2/OIDC authentication with role mapping and automatic project assignment, enabling seamless enterprise onboarding.
8. **Platform Features Delivery:** Built core product capabilities including environment management, deployment pipelines, analytics dashboards, release notes, and global search.
9. **DevOps & Infrastructure Automation:** Developed CI/CD pipelines with blue-green deployments, Docker-based environments, and Ansible automation across multi-cloud setups.

### 🏗️ Architecture

- **Control Plane:** Tenant lifecycle management, billing (Stripe), onboarding, and global configuration
- **Application Plane:** Multi-tenant runtime handling deployments, environments, and project-level operations
- **Deployment Models:** Shared (schema-based) for Free/Pro tiers, dedicated isolated stacks for Enterprise

### 🛠️ Tech Stack

SvelteKit, Spring Boot, PostgreSQL, RabbitMQ, Docker, Ansible, Stripe API, OAuth2/OIDC, Bitbucket Pipelines
